Searching in a descending sorted array

The method described in the theory performs binary search for arrays sorted in ascending order. Your task here is to modify the binarySearch method such that:

  • it searches in descending sorted arrays;
  • it returns the first index of a target element from the beginning of the array (the leftmost index);
  • it returns -1 if the given value is not present in the array.

Sample Input:
9
17 17 16 15 15 15 15 14 14
5
15 17 20 16 0

Sample Output:
3 0 -1 2 -1
